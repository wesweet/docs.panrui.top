#### 使用 html2canvas 生成图片问题

> 1：图片跨域问题(后端使图片能够进行跨域、前端图片标签添加 crossorigin="anonymous"属性)

> 2：二次生成榜单问题(所有图片 src 设置时添加时间戳后缀)

> 3: 使用 html2canvas 生成的元素不能存在滚动条，否则会导致生成的图片不完整。解决方案就是再父元素设置滚动条
